Our Mission and Philosophy

Code Differently aims to bridge the tech access gap by equipping the community with the technical and cognitive skills they’ll need to excel in technology-driven classrooms and workplaces. Through hands-on instruction and collaborative projects, Code Differently’s courses and events are designed to challenge, entertain and empower a community of learners while preparing them for an ever-evolving business landscape.
